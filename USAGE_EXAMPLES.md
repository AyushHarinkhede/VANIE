## 🆘 Common Questions

**Q: Can VANIE remember previous conversations?**
A: Current version doesn't persist data, but you can extend it with database.

**Q: Is VANIE always online?**
A: Yes, as long as the backend server is running.

**Q: Can I customize responses?**
A: Yes! Edit response_patterns in VANIE_FIXED.py

**Q: How do I add new features?**
A: Add new intents and handle them in the detect_intent() function.

**Q: Can I use this offline?**
A: Frontend works offline with fallback responses.

## 🌟 About VANIE

**VANIE** (Versatile AI for Natural Interaction and Enhancement) is a sophisticated, multilingual AI chatbot designed to provide intelligent, empathetic, and comprehensive conversational experiences. Built with advanced Natural Language Processing capabilities and an extensive knowledge base, VANIE can engage in meaningful conversations across 29+ different domains.

## 📋 Installation & Setup

### Prerequisites
- Python 3.8+
- Modern Web Browser (Chrome, Firefox, Safari, Edge)
- Git (for cloning)

### Quick Start
```bash
# Clone the repository
git clone https://github.com/AyushHarinkhede/VANIE.git

# Navigate to the project directory
cd VANIE

# Run the chatbot
python VANIE.py

# Open the web interface
# Open VANIE.html in your preferred web browser
```

### Dependencies
```bash
# Install required Python packages
pip install nltk spacy textblob

# Download NLTK data
python -m nltk.downloader all

# Download spaCy model
python -m spacy download en_core_web_sm
```
